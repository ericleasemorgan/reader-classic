Instructions

Distant Reader offers several ways to process corpora. Read our instructions below to learn how to view and create carrels with Distant Reader.

  • Overview
  • View Carrels
  • Create Carrels

Overview

Distant reading is an approach in literary studies that applies computational methods to literary data, usually derived from large digital libraries, for the purposes of literary history and theory. While the term is collective, and is used to refer to a range of different computational methods of analysing literary data, similar approaches also include macro-analysis, cultural analytics, computational formalism, computational literary studies, quantitative literary studies, and algorithmic literary criticism.

“Distant Reading” | Wikipedia

The Distant Reader is intended to supplement the traditional critical/analytical close-reading process, adding insights from quantitative analysis to make traditional qualitative reading processes more thorough and insightful. When given a corpus (a set of one or more texts), the Reader will summarize the results by outputting a set of structured data sets we call study carrels.

Since a study carrel is computer platform- and network-independent, the files in a study carrel are purposely designed to be be read by both people as well as just about any computer anywhere.

You may freely browse all public carrels, but to create a carrel, you must first sign in with your ORCID, Google, or institutional ID. Once you submit your collection for analysis, your carrel will be created. You will receive status updates by email during the carrel creation process.

Depending on the type and amount of content submitted, it can take as little as a few minutes, or as long as a few hours to create a study carrel. Typically, carrels are created in less than ten minutes.

Input

The Distant Reader can create a study carrel from a number of different input types, including:

  • One or more text (.txt) files
  • One or more comma-delimited (.csv) or tab-delimited files
  • One or more URLs
  • A .zip file, with an optional metadata file
  • A metadata file (.xml) from bioRxiv
  • A HathiTrust collection
  • A locally curated search subset of Project Gutenberg
  • A locally curated search subset of scholarly journals articles from the CORD-19 collection

Output

The Distant Reader outputs a large number of different files, such as:

  • An SQLite database file
  • A cache of the original documents
  • Narrative reports in both simple text and formatted HTML
  • A set of plain text files transformed from the items in the cache
  • Charts & graphs (and tab-delimited files) explicating:
    • Bibliographics
    • Parts-of-speech
    • Named entities and keywords
    • URLs
