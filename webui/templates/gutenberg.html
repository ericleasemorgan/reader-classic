GutenbergCarrel

Use this page to: 1) search a subset (30,000 items) of the venerable Project Gutenberg, 2) refine the results, and 3) create a study carrel from the list of found items.

The content of Project Gutenberg is strong on English literature, American literature, and Western philosophy, but just about any word or phrase will return something. This is a quick an easy way to create carrels whose content represents the complete works of a given author or an introduction a given subject. Enter a word, a few words, or a few words surrounded by quote marks to begin. The index supports an expressive query language, described in a blog posting. Stymied? Search for everything and use the resulting page to limit your query.
