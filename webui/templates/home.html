Intended for anybody who reads, the Distant Reader takes text as input, does analysis against it, and outputs sets of structured data, affectionately called "study carrels". The student, research, or scholar can then use a study carrel to read the text both closely as well as at a distance.

Working with Carrels

  • Browse Library of Carrels

    Browse the library of carrels you've created and those created and publicly shared by others.

  • Create a Carrel from Your Data

    Create a new carrel from a range of file types, including DOC/DOCX, PPT/PPTX, PDF, TXT, CSV, HTML and XML, and even URLs.

  • Create a Carrel from Project Gutenberg

    Create a carrel from a local cache of the approximately 30,000 public domain documents from Project Gutenberg.

  • Create a Carrel from CORD-19 Data

    Create a carrel from a cache of more than 100,000 scholarly and scientific journal articles on the topic of COVID-19.
