About This Project

Distant Reader is an open source software tool developed by Eric Lease Morgan at the University of Notre Dame. The Reader is supported by the work and feedback of many generous sponsors and contributors and includes the open source software libraries listed below.

Software packages

The Distant Reader makes use of the following software packages, according to each library's license.

  • Airavata
    Submit and monitor large-scale applications
  • Archive::Zip
    A Perl library to create Zip files
  • Bootstrap 4
    A front-end framework of web components
  • CGI
    A Perl module for creating CGI scripts
  • clipboard.js
    A Javascript library for copying DOM elements
  • csvkit
    A set of command-line tool reading tabular data
  • DBI
    A Perl library to do I/O against databases
  • File::Basename
    A Perl library to process file names
  • File::Copy
    A Perl library used to copy files
  • Gensim
    A Python library for natural language processing
  • HTML::Entities
    A Perl library for encoding data for the Web
  • jQuery
    A Javascript library for user-interface
  • JSON
    A Perl library for reading JSON data
  • mpl_toolkits
    A Python library for generating charts & graphs
  • networkx
    A Javascript library for creating network diagrams
  • nltk
    A Python library for natural language processing
  • OpenStack
    A Linux toolkit for creating virtual machines
  • pandas
    A Python library for processing matrix-based data
  • sklearn
    A Python library for machine learning
  • Slurm
    A Linux toolkit to support cluster computing
  • SpaCy
    A Python library for natural language processing
  • SQLAlchemy
    A Python library for interacting with databases
  • SQLite
    A C-based relational database
  • Textacy
    A Python library for natural language processing
  • Tika
    A Java archive for text pre-processing
  • WebService::Solr
    a Perl library used to interact with a Solr index
  • Wget
    A HTTP user-agent
  • wordcloud
    A Python library for creating word clouds
  • XML::XPath
    A Perl library for querying XML
